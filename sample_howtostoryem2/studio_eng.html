# Instructions
Based on the following story outline, output both Rule CSV and Scenario CSV in CSV format according to the output rules.

# Rule CSV Output Rules
Column: Content
ttl: Game Title (Displayed at the top of the screen)
bg: Initial background image filename
max_hp: Player's Maximum HP
init_skill: Initial Skill level
init_luck: Initial Luck level
init_food: Initial Food count

- Images are assumed to be in the same folder as the HTML; write filenames accurately with extensions.
- Use half-width alphanumeric characters for numbers.
- Comma-separated values.
- The first line must always be the column headers.

# Scenario CSV Output Rules
Column: Content
id: Paragraph ID (Integer)
type: normal / battle / end
text: Text to be displayed
options: Choices for the player
e_name: Enemy name (for 'battle' type)
e_skill: Enemy Skill level
e_hp: Enemy HP
next: Target paragraph ID after winning a battle
bgm: BGM filename
bgi: Paragraph-specific background image filename
effect: Special effects/Automatic triggers

# CSV Formatting Rules

▼ Types (type)
1. normal: Standard scenes.
Example: 1,normal,"The path splits.","Left→2,Right→3",,,,,bgm.mp3,room.jpg,

2. battle: Combat scenes.
Example: 2,battle,"An enemy attacks!","Fight",EnemyName,6,6,3,bgm_fight.mp3,battle.jpg,

3. end: Ending scenes.
Example: 10,end,"Game Over.","Finish",,,,,bgm_title.mp3,,

▼ Options (options)
Format: Display Text→Target ID
Multiple: Choice A→2,Choice B→3
Note: Always use the full-width arrow "→". Target IDs must exist.

▼ Text (text)
Do not use line breaks for AI generation.
When editing manually, you can use "\n" for line breaks.

▼ Combat Mechanics
Calculated as: 2d6 + Skill
- Win → Deal 2 damage to enemy
- Lose → Take 2 damage to self
- Using Luck: Modifies damage, but Luck decreases by 1 per use.

▼ Special Effects (effect)
Adds automatic effects to paragraphs. If HP reaches 0, it's Game Over.

Status Changes (Triggers on first visit):
- hp+3, hp-2, food+1, luck-1

Repeatable Effects (Add "!" prefix):
- !hp-2 (e.g., for traps or environmental damage)
Example: 11,normal,"A trap!","Go back→1",,,,,,,!hp-2

Healing / Eating:
- "eat": One-time meal point. Restores 4 HP (consumes 1 food).
- "!eat": Repeatable meal point.
Example: 4,normal,"A safe place to eat.","Proceed→5",,,,,,,!eat

▼ BGM
Enter the .mp3 filename. Leave blank to stop music.
