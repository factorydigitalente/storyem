# 命令
以下の物語の概要をもとに、CSV出力ルールに従ってCSV形式でルールCSVとシナリオCSVを出力してください。

# ルールCSVの出力ルール
項目名 内容
ttl ゲームタイトル（画面上部に表示）
bg 初期背景画像ファイル名
max_hp プレイヤー最大HP
init_skill 初期戦技
init_luck 初期幸運
init_food 初期食料
theme 色テーマ番号（1～9の整数）

▼theme（色テーマ）の種類
物語の雰囲気に合わせて 1～9 の番号を1つ選んでください。
1: 黒・グレー（標準・無機質・無彩色）
2: 赤（情熱・危機・炎）
3: 黄・金（砂漠・財宝・高貴）
4: 緑（森林・休息・自然）
5: 深緑（深い森・迷宮）
6: 青（深海・夜・冷静）
7: 紫（魔法・神秘・毒）
8: 茶（土・遺跡・レトロ）
9: 濃茶（泥・腐敗・絶望）

・画像はHTMLと同じフォルダに置くものとし、ファイル名は拡張子を含めて正確に書くものとする
・数値は半角数字
・カンマ区切り
・1行目は必ず項目名

# シナリオCSVの出力ルール
列 内容
id パラグラフ番号（整数）
type normal / battle / end
text 表示する文章
options 選択肢
e_name 敵の名前（battle時）
e_skill 敵の戦技
e_hp 敵HP
next 戦闘勝利後の移動先パラグラフID
bgm BGMファイル名
bgi 背景画像ファイル名
effect 特殊効果

▼type の種類
① normal：
通常シーン。
例：
1,normal,"通路は左右に分かれている。","左→2,右→3",,,,,bgm.mp3,room.jpg,

② battle：戦闘シーン。
2,battle,"鉄の兵が襲いかかる。","戦う",オートマン,6,6,3,bgm_fight.mp3,battle.jpg,

③ end：エンディング
10,end,"静寂が戻った。","おわり",,,,,bgm_title.mp3,,

▼options（選択肢）の書き方
形式：　表示テキスト→移動先ID
複数ある場合：　左へ→2,右へ→3

注意：
・「→」は全角矢印
・カンマで区切る
・移動先は必ず存在するID

▼text の書き方
AIでの生成では改行をつけると事故につながるため、改行はしないこと。
手動でCSVを作成する場合は、「\n」を改行記号として使用できます。

▼戦闘の仕組み
戦闘は、2d6 + 戦技　で判定されます。
勝利 → 敵に2ダメージ
敗北 → 自分に2ダメージ
幸運使用で増減
幸運は使うたび1減少

▼effect（特殊効果）
特定のパラグラフに自動効果を付けられます。
効果によってHPが0になった場合は、その時点でゲームオーバーとなります。

ステータス増減の形式（通常は初回訪問時のみ）：
hp+3
hp-2
food+1
luck-1

先頭に「!」を付けると、訪れるたびに繰り返し効果が発動します：
!hp-2（罠など）

例：以下の場合、同じidを訪れるたびにhp-2となります。
11,normal,"罠の道だ！","戻る→1",,,,,,,!hp-2
例：以下の場合、同じidを訪れても、2回目以降はfoodは増えません。
31,normal,"干し肉を見つけた。","進む→4",,,,,,,food+1

食事ポイントとして「eat」を書くと、食料がある場合にHPを4回復できるボタンが表示されます。
「eat」は1度きり、「!eat」は食料がある限り何度でも使用可能です。
例：
4,normal,"ここは安全そうだ。食料を好きなだけ食べても構わない。","進む→5",,,,,,,!eat

▼BGMについて
mp3ファイル名を書く
空欄なら停止
同じBGMなら切り替わらない
例：
bgm_fight.mp3

▼背景画像（bgi）
そのパラグラフの専用背景
空欄ならルールCSVで指定した背景を使用

★よくある記述ミス
・カンマ不足：列数が足りないと正常動作しません。すべての行は必ず列数が同じ（カンマの個数が同じ）であること（末尾が空でもカンマを打つ）。
・IDの重複：同じIDは使わないこと。
・存在しないIDへ移動しないこと
・options内の矢印は必ず「→」を使う。「->」などは禁止。
・戦闘行の整合性: battleタイプでは、5列目(敵名)から8列目(勝利後ID)までを必ず埋めること。
・引用符の管理: textとoptionsは必ず半角のダブルクォーテーション "" で囲むこと。

★その他
戦闘は少なめに。パラグラフ10個当たり2回がテンポ良い。
回復ポイントを作るとよい。
戦闘前後に eat を配置するとバランスが良い。
【B】ゲームブックとして作りたい物語の概要を入力してみましょう

例：スライムが魔王を倒す物語。戦闘は高難度で。
例：真冬の放課後恋愛ものをパラグラフ20個程度で。
例：富士登山（五合目までバス、八合目で宿泊、ご来光、砂走りで下山、麓で温泉と懐石料理）を戦闘なしで。
例：四択の選択式クイズを３問。
例：爆笑！闇鍋パーティー

※生成AI側の規約に反しないよう、良識の範囲内で入力しましょう。

もしも以前作成したルールCSVとシナリオCSVがあれば、それらの中身を上記の入力欄に貼り付け、さらに指示を文章で追記することで、改訂版を作ることもできます。（物語を膨らませたい・難易度を上げたいなど）
